# Privacy Policy — Gemini with Chen

**Last updated:** March 18, 2026

## Overview

Gemini with Chen is a Chrome extension that applies a custom visual theme to the Google Gemini web interface. This extension is purely cosmetic and does not collect, store, or transmit any user data.

## Data Collection

This extension does **not** collect any of the following:

- Personal information (name, email, address, etc.)
- Authentication credentials
- Browsing history
- Conversation content on Gemini
- Usage analytics or telemetry
- Cookies or tracking data
- Financial or payment information
- Location data

## Permissions

| Permission | Purpose |
|---|---|
| `host_permissions` (gemini.google.com) | Required to inject CSS and JavaScript that modify the visual appearance of the Gemini interface. |
| `storage` | Used to save user display preferences locally on the device. |

## Data Storage

The extension may store user display preferences (such as theme settings) locally on the device using `chrome.storage.local`. This data never leaves the device and is not transmitted to any server.

## Third-Party Services

This extension does not communicate with any external servers, APIs, or third-party services.
